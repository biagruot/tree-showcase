## Run Locally

Clone the project

```bash
  git clone https://github.com/biagruot/tree-showcase.git
```

Go to the project directory

```bash
  cd tree-showcase
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Run tests

```bash
  npm run test
```
